## Priority sources (immediate research)

Use these first before adding more tools to the pile.

| Priority | Source(s) | Use for |
|----------|-----------|---------|
| 1 | **Mobbin**, **Screenlane**, **Refero** | Real mobile flows: HQ, quest lists, onboarding, tabs |
| 2 | **Awwwards**, **Godly**, **Site of Sites** | Art direction: hero, type, poster energy (translate to mobile) |
| 3 | **Fontshare**, **Fonts in Use**, **Fontjoy** | Type pairing, metadata vs display, stamp/display contrast |
| 4 | **The Brand Identity**, **Pentagram**, **Logo Archive** | Brand system discipline — not copying identities |
| 5 | **Reanimated**, **Skia**, **LottieFiles**, **Jitter** | Motion language: stamp, reveal, scan, reward |
| 6 | **Rave-zines**, **Midwest rave fliers**, **David Ramsey Map Collection** | Dust & Iron texture; Story/map routing |
| 7 | **v0**, **shadcn**, **21st.dev**, **React Bits** | Component *ideas* — always rewrite for React Native |

### Secondary (after first 20 collected)

- **Cosmos / Are.na / Savee** — moodboards for approved direction
- **Design Spells / Before.click** — craft breakdowns
- **Tokens Studio + Figma** — token sync experiments (design ops)
- **MotionSites.ai / Emil Kowalski** — interaction quality bar
- **Meshgradient / Haikei** — abstract hero fallbacks (dev mockups only)
- **Milanote / Notion** — organize boards linked from reference board log

### De-prioritize for now

- **Three.js / Tripo / Autosprite** — until reward/3D scope is defined
- **Unsplash** — never ship in-app; moodboards only
- **Generic AI UI generators** — exploration only; log outputs, don't ship unreviewed
